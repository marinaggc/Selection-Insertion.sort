# Algoritmos de Ordenação: Selection Sort e Insertion Sort

Este projeto implementa dois algoritmos clássicos de ordenação: **Selection Sort** e **Insertion Sort**, com leitura de arquivos de entrada fornecidos pelo professor (no formato `.in`) e geração de arquivos de saída com os números ordenados.


## Como Executar


1. Execute o código com o nome do arquivo `.in` como argumento. Exemplo:

```bash
python ordena.py num.1000.1.in
